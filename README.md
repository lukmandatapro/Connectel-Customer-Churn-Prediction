# Connectel-Customer-Churn-Prediction
Customer churn prediction project focuses on analyzing customer behaviour and building a machine model to predict customers who are likely to leave. The goal is to help Connectel (a telecommunication company) to proactively identify at-risk customers and implement retention strategies to reduce churn rates

---
## Business Problem
Telecommunication companies faces significant financial losses when customer discontinue thier services. Understanding the factors that contributes to customer churn enables businesses to take preventive actions.

This projects aims to:
- Identify key fsctors driving customer churn
- Analyze customer behaviour pattern
- Build a predictive model to identify customers likely to churn

## Objectives
- Perform   exploratory data analysis (EDA) to understand customer pattern
- Identify variables influencing churn behaviour
- Engineer useful features foor modeling
- Train and evaluates machine learning models
- Provide insights and recommendations for customers

## Metodology
- Data collection and preprocessing: checked for missing values, handles missing data
- Exploratory data analysis: Performed univariate, Bivariate, Multivariate analysis
- Feature selection: created to inprove model performance
- Model developement: Model tested includes logistic regression, Random forest,Decision Tree classifiers, support vector classifiers etc
- Model evaluation and tuning: Model was evaluted using Accuracy, precision, Recall F1 Score, AUC/ROC value and comfusin matrix
- Interpretation and insight: Key factors influencing churn are Totalcharges, monthlycharges, tenure, contract type
- Insight: Cuustomers with Total charges and monthly billing showed higher churn probabilty

## Recommendations 
- Offer incentives to long term customer 
- Improve customer support responsive
